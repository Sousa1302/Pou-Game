# Pou Game (Python + Pygame)

A virtual pet game inspired by the classic **Pou**, built using Python and Pygame.
This project simulates a fully interactive pet with real-time state management, animations, and persistent data.

## Features

* Graphical interface using Pygame
* Feeding system
* Cleaning system
* Interactive gameplay
* Dynamic stats (hunger, happiness, etc.)
* Save & load system using JSON
* Random events and behaviors
* Real-time updates

## Technologies Used

* Python 3
* Pygame

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/Sousa1302/Pou-Game.git
```

2. Navigate to the project folder:

```bash
cd Pou-Game
```

3. Create a Python environment and install dependencies:

```bash
pip install pygame
```

4. Run the game:

```bash
python main.py
```
